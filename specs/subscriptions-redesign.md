# 訂閱功能重新設計

## 概述
重新設計訂閱功能，包含觀看頁面的訂閱按鈕和訂閱內容頁面的新版面設計。

## 需求

### 1. 觀看頁面訂閱按鈕
- [x] 在 YtWatch.vue 加入訂閱按鈕
- [x] 訂閱按鈕與頻道頁面效果一致
- [x] 顯示「訂閱」或「已訂閱」狀態

### 2. 訂閱內容頁面重新設計
- [x] 預設檢視模式：只顯示訂閱頻道卡片（網格排列）
- [x] 展開檢視模式：
  - 每個頻道為一個直向欄位（column）
  - 欄位最上方為頻道卡片
  - 下方為該頻道的影片列表（可垂直捲動）
  - 頻道欄位橫向排列
  - 支援橫向捲動瀏覽不同頻道
- [x] 提供切換檢視模式的按鈕

## 技術規格

### 相關檔案
- `src/renderer/views/YtWatch/YtWatch.vue` - 加入訂閱按鈕
- `src/renderer/views/YtSubscriptions/YtSubscriptions.vue` - 重新設計版面

## 驗收條件
- [x] 觀看頁面有訂閱按鈕，點擊可訂閱/取消訂閱
- [x] 訂閱內容頁面可切換卡片/展開檢視
