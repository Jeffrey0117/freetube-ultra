# FreeTube YouTube 風格 UI 改造規格

## 目標
讓 FreeTube Web 的首頁和影片頁面更像 YouTube 原生體驗。

---

## Part 1: 首頁改造

### 現狀問題
- 影片卡片排列不夠緊湊
- 縮圖比例可能不對
- 缺少 hover 效果
- 資訊排版不像 YouTube

### 目標樣式

```
┌─────────────────────────────────────────────────────────────┐
│ [縮圖 16:9 比例，hover 時顯示進度條預覽]                      │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │                                                         │ │
│ │                      影片縮圖                           │ │
│ │                                                 [12:34] │ │
│ └─────────────────────────────────────────────────────────┘ │
│ ┌──────┐  影片標題最多兩行...                               │
│ │ 頭像 │  頻道名稱                                         │
│ └──────┘  123萬次觀看 • 2週前                              │
└─────────────────────────────────────────────────────────────┘
```

### 具體改動

1. **縮圖**
   - 固定 16:9 比例
   - 圓角 12px
   - 右下角顯示影片長度 (黑底白字)
   - Hover 時輕微放大 (scale 1.02)

2. **標題**
   - 最多 2 行，超過顯示 ...
   - 字體加粗
   - 間距緊湊

3. **頻道資訊**
   - 左邊顯示小圓形頭像 (36px)
   - 頻道名稱 + 觀看次數 + 發布時間

4. **Grid 排列**
   - 響應式: 手機 1 欄, 平板 2-3 欄, 桌面 4-5 欄
   - 卡片間距 16px

## Part 2: 影片播放頁改造

### 現狀問題
- 播放器可能不夠大
- 資訊區排版
- 相關影片列表樣式

### 目標樣式 (桌面)

```
┌────────────────────────────────────────┬──────────────────┐
│                                        │                  │
│                                        │  [相關影片1]     │
│            影片播放器                   │  [相關影片2]     │
│            (16:9)                      │  [相關影片3]     │
│                                        │  [相關影片4]     │
├────────────────────────────────────────┤  [相關影片5]     │
│ 影片標題                               │                  │
│ 123萬次觀看 • 2024年1月1日             │                  │
├────────────────────────────────────────┤                  │
│ [頭像] 頻道名稱          [訂閱按鈕]    │                  │
│        123萬訂閱者                     │                  │
├────────────────────────────────────────┤                  │
│ 👍 1.2萬  👎  💬 留言  ➡️ 分享         │                  │
├────────────────────────────────────────┤                  │
│ 影片說明...                            │                  │
│ 顯示更多                               │                  │
└────────────────────────────────────────┴──────────────────┘
```

### 目標樣式 (手機)

```
┌─────────────────────────────────────────┐
│                                         │
│              影片播放器                  │
│              (16:9)                     │
│                                         │
├─────────────────────────────────────────┤
│ 影片標題                                │
│ 123萬次觀看 • 2024年1月1日              │
├─────────────────────────────────────────┤
│ [頭像] 頻道名稱              [訂閱]     │
├─────────────────────────────────────────┤
│  👍 1.2萬  │  👎  │  💬  │  ➡️ 分享    │
├─────────────────────────────────────────┤
│ 影片說明...  [顯示更多]                 │
├─────────────────────────────────────────┤
│ [相關影片 - 橫向滾動或直向列表]         │
└─────────────────────────────────────────┘
```

### 具體改動

1. **播放器**
   - 桌面: 左側佔 70% 寬度
   - 手機: 全寬，固定在頂部
   - 16:9 比例容器

2. **影片資訊區**
   - 標題大字 (20px)
   - 觀看數 + 日期 用較小灰色字
   - 頻道區: 頭像 (48px) + 名稱 + 訂閱數 + 訂閱按鈕

3. **互動按鈕**
   - 橫向排列: 讚、倒讚、留言、分享
   - YouTube 風格的膠囊按鈕

4. **相關影片**
   - 桌面: 右側欄，小縮圖 + 標題
   - 手機: 下方列表

## Part 3: 通用樣式

### 顏色 (深色模式)
```css
--bg-primary: #0f0f0f;
--bg-secondary: #272727;
--bg-hover: #3f3f3f;
--text-primary: #fff;
--text-secondary: #aaa;
--accent: #ff0000;
```

### 字體
```css
font-family: 'Roboto', 'Noto Sans TC', sans-serif;
```

### 過渡動畫
```css
transition: all 0.2s ease;
```
